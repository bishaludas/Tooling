# Tooling

## Vim Setup on Linux
- Clone the repo, copy the vimrc from vim folder into Home directory ('/') and rename it to .vimrc
- Install the vim plug package
```
https://github.com/junegunn/vim-plug
```

- Open vim and install the Plugins
```
:PlugInstall
```
